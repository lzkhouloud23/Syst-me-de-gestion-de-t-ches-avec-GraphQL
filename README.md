Système de gestion de tâches avec GraphQLCe projet est un système de gestion de tâches (To-Do List) développé dans le cadre du cours SoA et Microservices. Il utilise Node.js, Express, et Apollo Server pour implémenter une API GraphQL complète permettant de gérer des tâches.  
👤 InformationsAuteur: lzkhouloud23Classe: 4Info   Enseignant: Dr. Salah Gontara   Année Universitaire: 2025/2026 
🎯 Objectifs du TPComprendre la configuration de GraphQL avec Node.js et Express.  Créer un schéma GraphQL (typeDefs) et des résolveurs (resolvers).  Gérer les requêtes (Queries) et les mutations (Mutations) pour une API.
🛠️ Technologies UtiliséesNode.js: Environnement d'exécution.  Express: Framework web.  Apollo Server: Serveur pour traiter les requêtes GraphQL.  GraphQL: Langage de requête pour les API.
